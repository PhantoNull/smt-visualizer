# SMT visualizer

Visualizzatore web statico per esplorare una state machine a partire da `Broker XML` e `Kernel XML`.

## Live

[Apri la pagina GitHub Pages](https://phantonull.github.io/smt-visualizer/)

## Cosa fa

- renderizza stati, transizioni, self-loop e output
- supporta drag & drop dei nodi e posizioni salvate in sessione
- evidenzia reachability, stati finali e stati non raggiungibili
- include una `edit mode` per modificare stati e transizioni direttamente dal browser
- esporta il grafo in PNG

## Uso

1. Apri la pagina.
2. Incolla oppure carica `Broker XML` e `Kernel XML`.
3. Esplora il grafo, regola la spaziatura o attiva la modalità modifica.
